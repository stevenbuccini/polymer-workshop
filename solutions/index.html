Hello Polymer!

In this demo, you'll use your first Polymer element! But first, notice that we're working in a Bootstrap template. One of the neat things about Polymer is that since it's just plain ol' HTMLÂ®, it fits neatly into any existing stack or framework. Polymer and React? Sure, why not. Polymer and Bootstrap? No problem. The browser doesn't even know the difference, and neither will your users

Don't you just hate it when you're filling out a text input, but when you start typing the placeholder disappears and you forget what format you're supposed to use? Yeah, me neither. But just work with me here.

We can do better than that. Let's use Polymer's "paper-input" element, which has a neat floating label.

  1. First, install Polymer's paper-label web component:
    bower install --save Polymer/paper-input
  2. Next, load the element using HTML imports. Add the following snippet of code to your page's <head>:
    <link href="bower_components/paper-input/paper-input.html" rel="import">
  3. Now, add the element to the page by adding it in the HTML below:
    <paper-input floatingLabel label="Enter your email!"></paper-input>
  4. Note: Chrome's Same Origin Policy sucks, so loading this file locally won't work. Instead, in your terminal navigate to the directory where this file lives and run the command ython -m SimpleHTTPServer 8000. Now navigate to localhost:8000/

Congratulations! You just experienced the magic of Polymer for the first time. Notice you didn't have to do any hard work. The text box has smooth animations built-in. Pretty cool! This isn't great for all scenarios (i.e. this doesn't work in forms) but you can use the <paper-input-decorator> to wrap the normal HTML <input> to use in forms and to add validation.

Additional challenge problems:

  • Use <paper-input-decorator> to create a textbox that raises an error when you type an invalid email
  • After completing the above form, add it to a form where the submit button is a Polymer Paper component.
  • Package this form up into its own atomic component.

Creating your own Polymer element

Now, we're going to create our own Polymer element. Let's say our site has a gallery of business cards, one for each one of our team members. First, run the following command:
bower install --save Polymer/core-icon-button
Hop on over to business-card.html to begin!
